Aggregation #30

2016-04-24

Changes & Updates:

  • Added fallback for ‘say hi’ / ‘say hello’ without explicit target.
  • Added bridge, bridge, bridge, bridge, and bridge locations.
  • Added placeholders for alcove, split, pier, and circle locations.
  • Linked and added short descriptions for: circle, alcove, pier, and split locations. Can always get back to the bridge from these, but can only get to them from the bridge when configured correctly.
  • Added item placeholders: bedroll (in alcove), wheel (on bridge), mushrooms/flowers (in circle).
  • Installed conditional expression helper for Handlebars.
  • Added contextual bridge descriptions to vary by position (forest, underground, coast, void). Descriptions don’t make it very clear what’s happening, but the action of turning the wheel will.
  • Added wheel to bridge.
  • Stubbed in wheel behavior.
  • Added taxes module.
  • Added PAY TAXES command.
  • Added Devices module. Includes the device component and verbs for activating/rotating/manipulating. Devices can have a set of ‘states’ (on/off, rotational position, etc). By default, a device will cycle through the available states when triggered.
  • Wheel on bridge is now a device.
  • Designed my future office.
  • Wrote an introductory manual for the game.
  • Wrote up an expanded quest/puzzle line for the prologue, incorporating new areas (bridge, underground), the basin puzzle, and the Black Box objective. I think it will tie everything together into a more satisfying, game-like experience.
  • Made a character creator for a pen-and-paper RPG.
